void startTicToc ( )

Sets up the timer.

Must be called before any invocation to tic() or toc(), preferrably at the beginning of your application.

Here is the caller graph for this function:

void stopTicToc ( )

This function should be called when the time will not be being used any more. It destroys the events used to time CUDA kernels. If the timer is not running, this function does nothing and prints a warning message.

void tic ( )

Starts the timer.

Use toc() to get the elapsed time; tic() must be called before a toc().

Here is the caller graph for this function:

float toc ( )

Returns the elapsed time between its invocation and a previous invocation of toc(). Returns -1 and prints a warning message if toc() was not previously called. Returns -2 and prints and error message if start_tictoc() has not been called.

Returns
Elapsed time between tic() and toc() in milliseconds with a resolution of 0.5 microseconds.
